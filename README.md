## :muscle: 프로젝트 기획의도
- 헬스장 사용 회원들이 꾸준히 다닐 수 있도록 플랫폼 제공
- 온라인에서도 상담 및 회원권 결제 할 수 있도록 기능 제공
- 헬스장에 대한 정보 제공과 회원들간의 정보공유를 할 수 있는 커뮤니티 제공
